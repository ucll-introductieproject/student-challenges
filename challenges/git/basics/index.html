Git basics

Add, commit en push

  • Open vscode in de folder waar je challenges staan (bv. C:\introductieproject)

  • Ga naar de folder "git\basics"

  • Maak een nieuw bestand "solution.txt" en zet op de eerste lijn jouw naam en op de tweede lijn jouw studentnummer.

  • In Git bash voer je volgend commando uit:

$ git status

  • Zoals je kan zien detecteert git nu het bestand. Maar het is nog niet toegevoegd aan onze lokale git repository. Om dit te doen doe je:

$ git add solution.txt
$ git commit -m "Solution file"

  • Via "git add" zeg je tegen git om het bestand te tracken, "git commit" gaat het bestand effectief toevoegen aan de lokale repository. Via "-m" voeg je een boodschap toe aan de commit.

  • Ga nu via een browser naar je repository (plak de link die nog op je klembord zou moeten staan) en navigeer naar de folder "challenges/git/basics". Zoals je kan zien staat je solution.txt file daar nog niet. Dit komt omdat we de wijzigen die je net hebt gedaan op je lokale repository, moet "pushen" naar de remote versie:

$ git push

  • Wanneer je nu gaat refreshen in de browser zal je wel jouw solution file (en commit message) zien staan.

  • In git bash kan je het status commando nog eens uitvoeren. Normaal gezien zou de output als volgt moeten zijn:

Your branch is up to date with 'origin/master'.

nothing to commit, working tree clean

  • Probeer nu met wat je geleerd hebt een subfolder "Challenge_01" onder "git/basics" aan te maken. Plaats hierin een nieuw bestand "solution.txt" met de inhoud "Challenge 1 completed". Zorg ervoor dat deze folder in jouw remote repository komt.
